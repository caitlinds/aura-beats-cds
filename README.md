## :fire: Getting Started

<details open>
  <summary> How to Create a Mood </summary>
    1. Select "Create Mood" and type your desired mood along with a description. If you make a mistake, you can edit or delete the new mood. 
</details>

<details>
  <summary> How to Add a Video to Your Mood</summary>
    1. Select "Search Song" and type the song you would like to add to your mood playlist. 
    2. From the drop down menu, select the mood playlist to add the new video of the song. 
</details>

<details>
  <summary> How to View Your Moods</summary>
    1. Click "My Moods" to see a list of the moods you created. 
    2. Click on moods to see the associate playlist. 
    3. Add photos to your "My Moods" page by choosing file and uploading.  
</details>
